About this tool

Existing word counting solutions are poorly suited to LaTeX documents. General-purpose counters process raw source as plain text, producing inflated totals that include commands, preamble declarations, and environment syntax. TeXcount and Overleaf's built-in counter offer greater accuracy but present their own limitations: Overleaf's default "Total Words" counts only body text as interpreted by TeXcount, systematically excluding bibliography entries, citations, captions, and footnotes.

This tool performs single-pass parsing of LaTeX source entirely within the browser, requiring no server, no compilation step, and no external dependencies. A plain text mode is also available for counting essays and non-LaTeX documents.

Privacy

All word counting happens entirely within the browser. No source text is transmitted, uploaded, or stored on any server. The site uses Umami for anonymous, cookie-free analytics only.

Getting started

Paste your entire .tex file into the text area — preamble, \begin{document}, bibliography and all. Press Count Words or Ctrl+Enter.

LaTeX mode vs Plain text mode

By default the tool operates in LaTeX mode, which parses commands, environments, and comments (%). If you are pasting a plain essay (not LaTeX source), switch to Plain text mode. In plain text mode % signs are treated as literal characters and every word is counted directly.

Text inside thebibliography environment.

Verbose: Included / Excluded

Each individual word gets its own bordered highlight box — green for included, red for excluded — so you can see exactly what the counter treats as one “word”.
